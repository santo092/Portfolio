Crystals Collector Game

In the crystal collector game, the player's goal is to hit a target number by selecting various crystals. Each crystal has a hidden value and the player must select the crystals to add up to the target number. The player wins the game if they are able to reach the target number exactly, otherwise, they lose. Good luck!
